# 딥러닝 아키텍쳐(CNN, RNN, LSTM) 정리
## 1. CNN
- CNN(Convolutional Neural Network, 합성곱 신경망)은 주로 이미지나 영상 데이터를 처리할 때 사용하는 딥러닝 알고리즘의 한 종류입니다.
- 이미지의 공간적 구조(Spatial Structure)를 그대로 유지하면서 특징을 효과적으로 추출하도록 설계되었습니다.
- 특징 추출(Feature Extraction) 단계와 분류(Classification) 단계로 나뉩니다.
  - 합성곱 층(Convolutional Layer)
    - 이미지 위로 작은 필터를 일정 간격으로 이동시키며 연산을 수행합니다. 이 과정을 통해 이미지의 선, 면, 색상 등 주요 특징을 찾아내어 특정 맵을 생성합니다.
  - 풀링 층(Pooling Layer)
    - 특징 맵의 크기를 줄여 데이터의 양을 압축하는 과정입니다. 가장 큰 값을 뽑는 Max Pooling이 주로 사용되며, 이를 통해 사물의 위치가 조금 변해도 동일하게 인식할 수 있는 '불변성'을 확보하고 연산량을 줄입니다.
  - 완전 연결 층(Fully Connected Layer)
    - 특징 추출이 모두 끝나면 마지막에 데이터를 1차원으로 펼쳐서 출력층에 연결합니다. 여기서 최종적으로 이미지가 어떤 클래스에 속하는지 분류를 결정합니다.
- CNN의 작동 원리
  - 입력 : 원본 이미지 데이터가 들어옵니다.
  - 특징 추출 : 여러 겹의 합성곱 층과 풀링 층을 가지며 저수준 특징(점, 선)에서 고수준 특징(눈, 코, 입 등)을 단계별로 학습합니다.
  - 활성화 함수 : 보통 ReLU(Rectified Linear Unit) 함수를 사용하여 비선형성을 추가하고 학습 속도를 높입니다.
  - 최종 분류 : 추출된 고차원 특징들을 바탕으로 Softmax 등을 거쳐 최종 결과를 도출합니다.
- CNN의 장점
  - 파라미터 공유 : 전체 이미지를 한 번에 연결하는 방식보다 학습해야 할 파라미터 수가 훨씬 적어 호율적입니다.
  - 공간 정보 유지 : 픽셀 간의 연관성을 유지하므로 이미지 인식에서 압도적인 성능을 보입니다.
  - 다양한 활용 : 자율주행 자동차의 사물 인식, 의료 영상 분석, 얼굴 인식 기술 등 시각적 데이터가 쓰이는 거의 모든 분야의 핵심 기술입니다.
## 2. RNN
- RNN(Recurrent Neural Network, 순환 신경망)은 데이터 순서가 중요한 정보를 처리하기 위해 설계된 인공 신경망의 한 종류입니다.
- '기억'을 가지고 있어 시계열 데이터나 자연어 처리에 매우 강력한 성능을 발휘합니다.
- RNN의 핵심 구조
  - 은닉출ㄹ층(Hidden Layer)의 결과가 다시 자기 자신에게 입력됩니다.
  - 현재의 상태 : 현재 시점의 입력과 이전 시점의 상태를 함께 고려하여 결정됩니다.
  - 루프 구조 : 순환 구조 덕분에 과거의 정보가 현재의 결정에 영향을 줄 수 있습니다. 이를 통해 네트워크는 일종의 '단기 기억'을 갖게 됩니다.
- 데이터가 들어오는대로 계속 순환할 수 있기 때문에 입력과 출력의 길이에 제한이 없습니다.
- 매 시점마다 동일한 가중치와 편향을 사용 -> 데이터의 위치가 변하더라도 동일한 패턴을 인식할 수 있게 하며, 학습해야 할 파라미터 수를 대폭 줄여줍니다.
- 앞서 나온 정보를 바탕으로 현재 정보를 해석하므로, 문장의 흐름이나 주가의 변동 추이 같은 맥락을 파악하는 데 특화되어 있습니다.
- RNN은 문장이 너무 길어지면 아주 처음에 나왔던 정보를 뒤쪽까지 전달하지 못하고 잊어버리는 장기 의존성 문제가 있으며, 이를 보완하기 위해 정보의 보존 여부를 결정하는 게이트 구조를 추가한 모델들이 등장했습니다.
## 3. LSTM
- LSTM(Long Short-Term Memory, 장단기 메모리)은 기본 RNN의 치명적인 단점인 '기억력 부족' 문제를 해결하기 위해 고안된 특별한 종류의 순환 신경망입니다.
- LSTM의 핵심은 모듈의 윗부분을 가로지르는 셀 상태(Cell State)입니다.
  - 망각 게이트(Forget Gate)
    - 현재의 입력과 이전의 상태를 보고, 쓸모없다고 판단되는 정보는 삭제합니다.
  - 입력 게이트(Input Gate)
    - 새로 들어온 정보 중에서 셀 상태에 저장할 가치가 있는 중요한 정보를 골라냅니다.
  - 출력 게이트(Output Gate)
    - 업데이트된 셀 상태를 바탕으로, 현재 시점에서 어떤 값을 출력으로 내보낼지 결정합니다.
- 장점
  - 긴 시퀀스 처리 : RNN에 비해 훨씬 긴 데이터를 안정적으로 학습할 수 있습니다.
  - 성능 : 자연어 번역, 음성 인식, 주가 예측 등에서 한동안 표준 모델로 자리 잡았을 만큼 강력합니다.
- 단점
  - 복잡성 : 구조가 복잡해서 기본 RNN보다 연상량이 많고 학습 속도가 상대적으로 느립니다.
  - 병렬 처리의 어려움 : 데이터를 순서대로 처리해야 하므로 최신 기술인 Transformer 모델에 비해 대규모 데이터 학습 효율이 떨어집니다.
